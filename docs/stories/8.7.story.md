## Story

**As a** usuario que usa agentes IA en su sistema (activities con executor_type='ai'),
**I want** ver un reporte de apalancamiento de agentes que muestre cuántas tareas ejecutó la IA vs el humano y el ROI estimado,
**so that** pueda entender el valor que aportan los agentes IA a mi sistema y optimizar su uso.

## Acceptance Criteria

### AC1 — Función pura `computeLeverageMetrics`
**Given** un array de activities con `executorType`, `status`, y `scheduledDurationMinutes`,
**When** se llama a `computeLeverageMetrics(activities)`,
**Then** retorna:
```typescript
{
  totalActivities: number
  aiActivities: number
  humanActivities: number
  mixedActivities: number
  aiPercentage: number                    // aiActivities / totalActivities * 100
  aiCompletedSuccessfully: number         // ai con status='completed'
  aiAccuracyRate: number                  // aiCompletedSuccessfully / aiActivities (0-1)
  estimatedHumanHoursSaved: number        // suma scheduledDurationMinutes de ai completadas / 60
}
```

### AC2 — Función pura `computeROIByExecutorType`
**Given** activities agrupadas por executor_type y sus duraciones,
**When** se llama a `computeROIByExecutorType(activities)`,
**Then** retorna `{ executorType, count, totalMinutes, avgMinutes }[]` para cada tipo.

### AC3 — Server Action `getAgentLeverageReport(period)`
**Given** el usuario autenticado y un período,
**When** se llama a `getAgentLeverageReport(period)`,
**Then**:
1. Obtiene activities del período con `executorType`, `status`, `scheduledDurationMinutes`
2. Ejecuta `computeLeverageMetrics` y `computeROIByExecutorType`
3. Retorna el reporte completo

### AC4 — Umbral mínimo de datos (FR21)
**Given** el usuario con menos de 14 días de activities,
**When** se llama a `getAgentLeverageReport`,
**Then** incluye campo `sufficientData: false` con mensaje "Se necesitan al menos 14 días de datos para el reporte de apalancamiento."

### AC5 — UI: sección "Agent Leverage" en `/reports`

**When** se renderiza la sección,
**Then** aparece:
- Métrica grande: "X% ejecutado por IA" con desglose visual (barra IA/Humano/Mixto)
- AI Accuracy Rate: "Y% de tareas IA completadas sin corrección"
- Horas ahorradas estimadas: "~Z horas"
- Tabla de ROI por executor_type: tipo + cantidad + tiempo total

## Business Value

FR21 completo. El usuario ve el ROI tangible de los agentes IA. Si el sistema muestra "la IA ejecutó el 40% de tus tareas con 95% de accuracy y te ahorró 12 horas esta semana", el valor del sistema es incontestable. También identifica qué tipo de executor tiene mejor performance.

## Risks

- **`scheduledDurationMinutes` null:** Muchas activities pueden no tener duración planificada — estimar horas ahorradas solo para activities con duración definida.
- **Accuracy ambigua:** "Completado sin corrección" se aproxima con `status='completed'` — no hay campo de "requirió corrección" en MVP.
- **Sin activities AI:** Si el usuario no usa agentes, mostrar "No hay activities ejecutadas por IA en este período."

## Dev Notes

### `features/reports/leverage.ts`

```typescript
export interface LeverageMetrics {
  totalActivities: number
  aiActivities: number
  humanActivities: number
  mixedActivities: number
  aiPercentage: number
  aiAccuracyRate: number
  estimatedHumanHoursSaved: number
}

export function computeLeverageMetrics(
  activities: { executorType: 'human' | 'ai' | 'mixed'; status: string; scheduledDurationMinutes: number | null }[]
): LeverageMetrics {
  const total = activities.length
  const ai = activities.filter(a => a.executorType === 'ai')
  const human = activities.filter(a => a.executorType === 'human')
  const mixed = activities.filter(a => a.executorType === 'mixed')
  const aiCompleted = ai.filter(a => a.status === 'completed')
  const hoursSaved = aiCompleted.reduce((s, a) => s + (a.scheduledDurationMinutes ?? 0), 0) / 60
  return {
    totalActivities: total,
    aiActivities: ai.length,
    humanActivities: human.length,
    mixedActivities: mixed.length,
    aiPercentage: total > 0 ? (ai.length / total) * 100 : 0,
    aiAccuracyRate: ai.length > 0 ? aiCompleted.length / ai.length : 0,
    estimatedHumanHoursSaved: hoursSaved,
  }
}

export function computeROIByExecutorType(
  activities: { executorType: 'human' | 'ai' | 'mixed'; scheduledDurationMinutes: number | null }[]
): { executorType: string; count: number; totalMinutes: number; avgMinutes: number }[] {
  const groups = new Map<string, { count: number; totalMinutes: number }>()
  for (const a of activities) {
    const g = groups.get(a.executorType) ?? { count: 0, totalMinutes: 0 }
    groups.set(a.executorType, { count: g.count + 1, totalMinutes: g.totalMinutes + (a.scheduledDurationMinutes ?? 0) })
  }
  return Array.from(groups.entries()).map(([executorType, g]) => ({
    executorType,
    count: g.count,
    totalMinutes: g.totalMinutes,
    avgMinutes: g.count > 0 ? g.totalMinutes / g.count : 0,
  }))
}
```
