## Story

**As a** usuario planificando sus OKRs anuales,
**I want** ver un OKR Impact Score que me indique cuánto mejoraría mi Life System Health Score global si logro el OKR al 100%,
**so that** pueda priorizar objetivos que maximicen mi bienestar sistémico en lugar de solo los que más me gustan.

## Scope

**IN scope:**
- Lógica pura en `features/maslow/okr-impact.ts`:
  - `calculateOKRImpact(okrCandidate, areas, currentScores)` → `OKRImpactResult`
  - Simula el score global si el área vinculada al OKR sube al nivel objetivo (default: +20 puntos o 100%)
  - Usa los mismos multiplicadores Maslow de `lib/utils/maslow-weights.ts`
  - Calcula delta de Health Score: impacto = scoreGlobalSimulado − scoreGlobalActual
- Componente `OKRImpactBadge` — muestra el delta en formato "+X.X pts" con color según magnitud
- Integración del badge en `OKRForm` existente (`components/okrs/OKRForm.tsx`) al seleccionar área
- Tests unitarios: toda la lógica de `okr-impact.ts`
- TypeScript strict, sin `any`

**OUT of scope:**
- CRUD completo de OKRs (Story 3.5-3.6)
- Impact Score para KRs trimestrales (solo OKRs anuales en MVP)
- Persistencia del Impact Score en DB (calculado en runtime)
- Ranking automático de OKRs por impacto
- Comparación multi-OKR simultánea

## Acceptance Criteria

1. `calculateOKRImpact(okrCandidate, areas, scoreMap)` retorna `OKRImpactResult` correctamente tipado
2. OKR vinculado a área nivel 1 (peso 2.0×) tiene mayor impacto potencial que área nivel 7 (peso 1.0×) dado el mismo delta de score
3. Si el área ya tiene score = 100 → impacto = 0
4. Si `areaId` es null o área no encontrada → impacto = 0
5. El delta se calcula correctamente: `globalSimulado(scoreArea=targetScore) − globalActual`
6. `OKRImpactBadge` muestra "+X.X pts" en verde si impacto > 2, amarillo si 0-2, gris si 0
7. Badge visible en `OKRForm` al seleccionar un área vinculada
8. Badge oculto cuando no hay área seleccionada o impacto = 0 con área ya optimizada
9. `npm run typecheck` PASS, `npm run lint` PASS (0 errores)
10. Tests unitarios PASS: todos los casos de `calculateOKRImpact`

## Dev Notes

### Insights Técnicos y Referencias

- **Data Models:** Reutilizar `MaslowLevel` (1-8) y los pesos de `lib/utils/maslow-weights.ts` [Source: docs/architecture/source-tree.md#lib/utils/maslow-weights.ts].
- **Cálculo del Scoring:** La fórmula oficial para el Life System Health Score es `Σ(score_área × multiplicador_nivel) / Σ(multiplicadores)` [Source: docs/prd/technical-assumptions.md#cálculo-del-life-system-health-score].
- **Estructura de Pesos:** D-Needs críticos (1-2) tienen peso 2.0x, mientras que B-Needs avanzados (7-8) tienen peso 1.0x [Source: docs/prd/technical-assumptions.md#multiplicadores-de-peso-life-system-health-score].
- **Ubicación de Archivos:** La lógica debe residir en `features/maslow/` y los tests en `tests/unit/maslow/` espejando la estructura [Source: docs/architecture/source-tree.md#tests/].
- **Estrategia de Tests:** Foco en tests unitarios con Vitest para la lógica pura de scoring, buscando un coverage mínimo del 80% [Source: docs/architecture/test-strategy.md#13.2-unit-tests-vitest].

### Tipos

```typescript
// features/maslow/okr-impact.ts
export interface OKRCandidate {
  areaId: string | null
  targetScore?: number  // score objetivo si OKR completado al 100% — default 100
}

export interface OKRImpactResult {
  deltaPoints: number       // diferencia en Health Score global (puede ser 0)
  currentAreaScore: number  // score actual del área vinculada
  targetAreaScore: number   // score objetivo simulado
  areaWeight: number        // multiplicador Maslow del área
}
```

### Algoritmo de simulación

```typescript
// 1. Obtener score actual del área vinculada desde scoreMap
// 2. targetScore = okrCandidate.targetScore ?? 100
// 3. Calcular globalActual usando calculateGlobalScore(scoreMap)
// 4. Crear scoreMapSimulado = { ...scoreMap, [areaId]: targetScore }
// 5. Calcular globalSimulado usando calculateGlobalScore(scoreMapSimulado)
// 6. deltaPoints = globalSimulado - globalActual

// Reutilizar directamente:
import { calculateGlobalScore } from '@/features/maslow/scoring'
import { MASLOW_WEIGHTS, MASLOW_TOTAL_WEIGHT } from '@/lib/utils/maslow-weights'
import type { MaslowLevel } from '@/lib/utils/maslow-weights'
```

### Nota sobre calculateGlobalScore

La función `calculateGlobalScore` en `features/maslow/scoring.ts` acepta
`scoreMap: Record<MaslowLevel, number>` donde MaslowLevel es 1-8.
El `scoreMap` se construye mapeando `area.maslowLevel → area.currentScore`.

### OKRImpactBadge — estilos

```
deltaPoints > 2  → text-green-600  bg-green-50  "+X.X pts"
deltaPoints 0-2  → text-yellow-600 bg-yellow-50 "+X.X pts"
deltaPoints = 0  → oculto (no renderizar si área ya en 100% o sin área)
```
