## Story

**As a** usuario que registra actividades en el calendario,
**I want** que el sistema detecte automáticamente patrones de comportamiento recurrentes que no he registrado como hábitos formales,
**so that** pueda tomar conciencia de hábitos que ya tengo pero no he reconocido explícitamente.

## Acceptance Criteria

### AC1 — Función pura `detectUnconsciousHabits`
**Given** un array de activities con título, área y `scheduledAt`, más los hábitos ya registrados del usuario,
**When** se llama a `detectUnconsciousHabits(activities, existingHabitNames)`,
**Then** retorna sugerencias `{ term, occurrences, avgIntervalDays, consistency }[]` para patterns que:
- Aparecen en ≥ 7 activities distintas en los últimos 90 días
- Tienen un intervalo promedio entre ocurrencias de ≤ 14 días (frecuencia semanal/quincenal)
- No existen ya como hábito (case-insensitive match por nombre)
- El término tiene > 3 caracteres y no es stop word

### AC2 — Solo retorna patrones con consistencia ≥ 0.5
**Given** activities recurrentes,
**When** el patrón tiene menos del 50% de regularidad (ej: ocurre aleatoriamente sin patrón claro),
**Then** NO aparece en las sugerencias.
`consistency = 1 - (stdDev(intervals) / mean(intervals))` — 1.0 = perfectamente regular.

### AC3 — Server Action `detectUnconsciousHabitsFromActivities()`
**Given** el usuario autenticado,
**When** se llama a `detectUnconsciousHabitsFromActivities()`,
**Then** obtiene activities de los últimos 90 días con `scheduledAt` y ejecuta `detectUnconsciousHabits`.
Retorna array de sugerencias (puede ser vacío).

### AC4 — Server Action `createHabitFromPattern(term, areaId?)`
**Given** una sugerencia detectada,
**When** el usuario confirma,
**Then** crea un hábito usando la acción existente de hábitos y retorna `{ success: true, habitId }`.

### AC5 — Server Action `dismissUnconsciousHabit(term)`
**Given** una sugerencia mostrada,
**When** el usuario descarta,
**Then** persiste el descarte en localStorage (MVP) para no volver a sugerir.

### AC6 — UI: banner "Hábitos detectados" en `/habits`
**Given** `detectUnconsciousHabitsFromActivities` retorna sugerencias,
**When** se renderiza `/habits`,
**Then** aparece banner con cards por sugerencia:
- Término detectado
- "X veces en 90 días · cada ~Y días"
- Botón "✓ Crear hábito" (con selector de área)
- Botón "Ignorar"

### AC7 — Tests ≥ anteriores + nuevos de Story 8.5
**Given** la implementación completa,
**When** se ejecuta `npm test`,
**Then** todos los tests pasan.

## Business Value

Cierra el ciclo de detección automática (detectEmergingSkills para Skills, detectUnconsciousHabits para Hábitos). El usuario descubre comportamientos que ya tiene pero no ha formalizado — aumenta el autoconocimiento sin esfuerzo manual.

## Risks

- **Intervalo irregular:** Si el usuario hace algo 7 veces pero de forma completamente aleatoria, `consistency < 0.5` lo filtra correctamente.
- **Coincidencia con hábitos existentes:** Comparación case-insensitive por nombre base (tokenización) para evitar sugerir lo que ya existe.
- **Stop words:** Reutilizar STOP_WORDS de `features/skills/detection.ts` + añadir términos específicos de hábitos.

## Dev Notes

### `detectUnconsciousHabits` sketch

```typescript
// features/habits/unconscious-detection.ts
// Similar pattern to detectEmergingSkills (Story 7.3)

export interface UnconsciousHabitSuggestion {
  term: string
  occurrences: number
  avgIntervalDays: number
  consistency: number  // 0-1
}

// Algorithm:
// 1. Tokenize activity titles (same as detection.ts: lowercase, strip, split, filter stop words)
// 2. For each term: collect all scheduledAt dates where the term appears
// 3. Compute intervals between consecutive dates
// 4. consistency = 1 - (stdDev(intervals) / mean(intervals)) — capped 0-1
// 5. Filter: occurrences >= 7 AND consistency >= 0.5 AND avgIntervalDays <= 14
// 6. Exclude terms matching existing habit names

const HABIT_MIN_OCCURRENCES = 7
const HABIT_MAX_INTERVAL_DAYS = 14
const HABIT_MIN_CONSISTENCY = 0.5
```

### Stddev helper

```typescript
function stdDev(values: number[]): number {
  if (values.length < 2) return 0
  const mean = values.reduce((s, v) => s + v, 0) / values.length
  return Math.sqrt(values.reduce((s, v) => s + (v - mean) ** 2, 0) / values.length)
}
```
