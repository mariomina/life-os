## Story

**As a** usuario que ejecuta actividades desde el calendario,
**I want** iniciar, pausar (con razón) y detener un timer para cada actividad,
**so that** el sistema calcule automáticamente el tiempo real invertido sin que yo tenga que estimarlo manualmente.

## Acceptance Criteria

### AC1 — Botón "Iniciar" inicia timer en DayView

**Given** el usuario está en la Vista Día y ve una actividad pendiente,
**When** hace clic en el botón "Iniciar",
**Then:**
- Se llama el Server Action `startTimer(activityId)` que inserta una fila en `time_entries`:
  - `stepActivityId` = activityId del evento
  - `userId` = usuario autenticado (desde `supabase.auth.getUser()`)
  - `startedAt` = timestamp actual con timezone (UTC)
  - `endedAt` = null
  - `durationSeconds` = null
  - `isActive` = true
- Si ya existe un timer activo (`isActive=true`) para esta actividad: retorna error "Ya hay un timer activo para esta actividad"
- Si el Server Action es exitoso: el botón "Iniciar" cambia a "Detener" + "Pausar" para esa actividad
- `revalidatePath('/calendar')` se llama tras la inserción exitosa

### AC2 — Botón "Detener" calcula y persiste duración real

**Given** el usuario tiene un timer activo (isActive=true) para una actividad,
**When** hace clic en "Detener",
**Then:**
- Se llama el Server Action `stopTimer(entryId)` que actualiza la fila en `time_entries`:
  - `endedAt` = timestamp actual con timezone (UTC)
  - `durationSeconds` = EXTRACT(EPOCH FROM now() - started_at) redondeado a entero, descontando tiempo pausado si `pausedAt` estaba seteado
  - `isActive` = false
- El botón vuelve a mostrar "Iniciar" para esa actividad
- El tiempo total acumulado de la actividad = `SUM(durationSeconds)` WHERE `stepActivityId = activityId`
- `revalidatePath('/calendar')` se llama tras la actualización exitosa

### AC3 — Pausar timer con razón obligatoria

**Given** el usuario tiene un timer activo para una actividad,
**When** hace clic en "Pausar",
**Then:**
- Se muestra un prompt/input inline para ingresar la razón de la pausa (campo de texto, requerido, max 200 chars)
- Si el usuario confirma con una razón: se llama `pauseTimer(entryId, reason)` que actualiza:
  - `pausedAt` = timestamp actual con timezone (UTC)
  - `pauseReason` = razón ingresada (trimmed)
  - `isActive` = false (timer en pausa)
- Si el usuario cancela el prompt: no se pausa (el timer sigue corriendo)
- El botón "Pausar" cambia a "Reanudar" mientras el timer está pausado

### AC4 — Reanudar timer pausado

**Given** el usuario tiene un timer pausado (`isActive=false`, `pausedAt` set) para una actividad,
**When** hace clic en "Reanudar",
**Then:**
- Se llama `resumeTimer(entryId)` que inserta una **nueva fila** en `time_entries` con:
  - `stepActivityId` = mismo stepActivityId
  - `userId` = usuario autenticado
  - `startedAt` = timestamp actual (nueva sesión de tiempo)
  - `isActive` = true
- La fila anterior (la pausada) queda con `isActive=false`, `pausedAt` y `pauseReason` preservados
- El botón vuelve a mostrar "Detener" + "Pausar"
- `revalidatePath('/calendar')` tras la inserción

### AC5 — Duración total acumulada visible en DayView

**Given** una actividad tiene una o más entradas en `time_entries` con `durationSeconds` calculados,
**When** el usuario ve la Vista Día,
**Then:**
- Cada evento muestra el tiempo acumulado en formato legible (ej. "45 min", "1h 20min") si `totalSeconds > 0`
- El tiempo se calcula como `SUM(durationSeconds)` de todas las entradas con `durationSeconds IS NOT NULL` para ese `stepActivityId`
- Si no hay tiempo registrado (actividad sin entradas): no se muestra el tiempo (no muestra "0 min")
- El dato se obtiene desde la query inicial en `page.tsx` (Server Component), no en tiempo real (eso es Story 5.9)

### AC6 — Server Actions en `actions/timer.ts`

**Given** la implementación de time tracking,
**When** se revisa el código,

### Server Actions — `actions/timer.ts`

```typescript
'use server'

import { eq, and, isNull, sum } from 'drizzle-orm'
import { revalidatePath } from 'next/cache'
import { createSupabaseServerClient } from '@/lib/supabase/server'
import { db, assertDatabaseUrl } from '@/lib/db/client'
import { timeEntries } from '@/lib/db/schema/time-entries'

export interface TimerActionResult {
  error: string | null
  entryId?: string
}

const UUID_REGEX = /^[0-9a-f]{8}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{12}$/i

async function getAuthenticatedUserId(): Promise<string> {
  const supabase = await createSupabaseServerClient()
  const { data: { user } } = await supabase.auth.getUser()
  if (!user) throw new Error('UNAUTHENTICATED')
  return user.id
}

export async function startTimer(activityId: string): Promise<TimerActionResult> {
  if (!activityId || !UUID_REGEX.test(activityId)) return { error: 'ID de actividad inválido' }
  try {
    assertDatabaseUrl()
    const userId = await getAuthenticatedUserId()
    // Verificar que no haya timer activo para esta actividad
    const existing = await db
      .select({ id: timeEntries.id })
      .from(timeEntries)
      .where(and(eq(timeEntries.stepActivityId, activityId), eq(timeEntries.isActive, true)))
      .limit(1)
    if (existing.length > 0) return { error: 'Ya hay un timer activo para esta actividad' }
    const [entry] = await db.insert(timeEntries).values({
      stepActivityId: activityId,
      userId,
      startedAt: new Date(),
      isActive: true,
    }).returning({ id: timeEntries.id })
    revalidatePath('/calendar')
    return { error: null, entryId: entry.id }
  } catch (err) {
    console.error('[startTimer] failed:', err)
    return { error: 'No se pudo iniciar el timer. Inténtalo de nuevo.' }
  }
}

export async function stopTimer(entryId: string): Promise<TimerActionResult> {
  if (!entryId || !UUID_REGEX.test(entryId)) return { error: 'ID de entrada inválido' }
  try {
    assertDatabaseUrl()
    const userId = await getAuthenticatedUserId()
    const [entry] = await db
      .select()
      .from(timeEntries)
      .where(and(eq(timeEntries.id, entryId), eq(timeEntries.userId, userId)))
      .limit(1)
    if (!entry) return { error: 'Entrada de timer no encontrada' }
    const now = new Date()
    const startMs = entry.startedAt.getTime()
    const durationSeconds = Math.round((now.getTime() - startMs) / 1000)
    await db.update(timeEntries)
      .set({ endedAt: now, durationSeconds, isActive: false, updatedAt: now })
      .where(and(eq(timeEntries.id, entryId), eq(timeEntries.userId, userId)))
    revalidatePath('/calendar')
    return { error: null }
  } catch (err) {
    console.error('[stopTimer] failed:', err)
    return { error: 'No se pudo detener el timer.' }
  }
}

export async function pauseTimer(entryId: string, reason: string): Promise<TimerActionResult> {
  if (!entryId || !UUID_REGEX.test(entryId)) return { error: 'ID de entrada inválido' }
  const trimmedReason = reason.trim()
  if (!trimmedReason) return { error: 'La razón de la pausa es requerida' }
  if (trimmedReason.length > 200) return { error: 'La razón no puede superar 200 caracteres' }
  try {
    assertDatabaseUrl()
    const userId = await getAuthenticatedUserId()
    const now = new Date()
    await db.update(timeEntries)
      .set({ pausedAt: now, pauseReason: trimmedReason, isActive: false, updatedAt: now })
      .where(and(eq(timeEntries.id, entryId), eq(timeEntries.userId, userId)))
    revalidatePath('/calendar')
    return { error: null }
  } catch (err) {
    console.error('[pauseTimer] failed:', err)
    return { error: 'No se pudo pausar el timer.' }
  }
}

export async function resumeTimer(entryId: string): Promise<TimerActionResult> {
  if (!entryId || !UUID_REGEX.test(entryId)) return { error: 'ID de entrada inválido' }
  try {
    assertDatabaseUrl()
    const userId = await getAuthenticatedUserId()
    // Obtener stepActivityId de la entrada pausada
    const [paused] = await db
      .select({ stepActivityId: timeEntries.stepActivityId })
      .from(timeEntries)
      .where(and(eq(timeEntries.id, entryId), eq(timeEntries.userId, userId)))
      .limit(1)
    if (!paused) return { error: 'Entrada de timer no encontrada' }
    // Insertar nueva entrada activa (nueva sesión de tiempo)
    const [newEntry] = await db.insert(timeEntries).values({
      stepActivityId: paused.stepActivityId,
      userId,
      startedAt: new Date(),
      isActive: true,
    }).returning({ id: timeEntries.id })
    revalidatePath('/calendar')
    return { error: null, entryId: newEntry.id }
  } catch (err) {
    console.error('[resumeTimer] failed:', err)
    return { error: 'No se pudo reanudar el timer.' }
  }
}
```

### Query tiempo acumulado — `page.tsx`

```typescript
// Obtener tiempo acumulado por actividad para el rango visible
// Retorna: Record<stepActivityId, totalSeconds>
async function getTimeTotalsForActivities(
  userId: string,
  activityIds: string[]
): Promise<Record<string, number>> {
  if (activityIds.length === 0) return {}
  const rows = await db
    .select({
      stepActivityId: timeEntries.stepActivityId,
      total: sum(timeEntries.durationSeconds),
    })
    .from(timeEntries)
    .where(
      and(
        eq(timeEntries.userId, userId),
        inArray(timeEntries.stepActivityId, activityIds),
        isNotNull(timeEntries.durationSeconds)
      )
    )
    .groupBy(timeEntries.stepActivityId)
  return Object.fromEntries(
    rows.map((r) => [r.stepActivityId, Number(r.total ?? 0)])
  )
}
```

interface TimerState {
  entryId: string
  isPaused: boolean
}
// activeTimers: Map<activityId, TimerState>
// Se inicializa desde props que page.tsx provee (query de timeEntries activos)
// Al hacer startTimer/stopTimer/pauseTimer/resumeTimer → actualiza el Map optimisticamente
// revalidatePath del servidor sincroniza el estado real

// Funciones en CalendarClient:
async function handleStartTimer(activityId: string) {
  const { startTimer } = await import('@/actions/timer')
  const result = await startTimer(activityId)
  if (result.error) { showToast(`Error: ${result.error}`); return }
  if (result.entryId) {
    setActiveTimers(prev => new Map(prev).set(activityId, { entryId: result.entryId!, isPaused: false }))
  }
}

async function handleStopTimer(activityId: string, entryId: string) {
  const { stopTimer } = await import('@/actions/timer')
  const result = await stopTimer(entryId)
  if (result.error) { showToast(`Error: ${result.error}`); return }
  setActiveTimers(prev => { const m = new Map(prev); m.delete(activityId); return m })
}

### Formato de tiempo acumulado

```typescript
export function formatSeconds(totalSeconds: number): string {
  if (totalSeconds <= 0) return ''
  const h = Math.floor(totalSeconds / 3600)
  const m = Math.floor((totalSeconds % 3600) / 60)
  if (h > 0) return m > 0 ? `${h}h ${m}min` : `${h}h`
  return `${m} min`
}
```
