## Story

**As a** usuario que completó el diagnóstico inicial (via cuestionario o upload),
**I want** ver una Vista Perfil en el Home con mis 8 scores de áreas Maslow agrupados en D-Needs y B-Needs, mi Life System Health Score global y la tendencia de cada área,
**so that** tenga una visión clara de mi baseline y pueda entender en qué áreas necesito enfoque.

## Dev Notes

### Tendencia desde scores históricos

```typescript
// lib/utils/trend.ts
export function calculateTrend(scores: number[]): '↑' | '↓' | '→' {
  if (scores.length < 2) return '→'
  const last = scores[scores.length - 1]
  const prev = scores[scores.length - 2]
  if (last > prev) return '↑'
  if (last < prev) return '↓'
  return '→'
}
```

### Query de areas con Drizzle

```typescript
// lib/db/queries/areas.ts
import { eq, asc } from 'drizzle-orm'
import { db } from '@/lib/db/client'
import { areas } from '@/lib/db/schema'

export async function getUserAreas(userId: string) {
  return db
    .select()
    .from(areas)
    .where(eq(areas.userId, userId))
    .orderBy(asc(areas.maslowLevel))
}
```

### Query de scores históricos

```typescript
// lib/db/queries/area-scores.ts
import { eq, desc } from 'drizzle-orm'
import { db } from '@/lib/db/client'
import { areaScores } from '@/lib/db/schema'

export async function getRecentAreaScores(userId: string, limit = 7) {
  return db
    .select()
    .from(areaScores)
    .where(eq(areaScores.userId, userId))
    .orderBy(desc(areaScores.scoredAt))
    .limit(limit * 8) // 8 areas × N days
}
```
