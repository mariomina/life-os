## Story

**As a** usuario que registra tiempo en actividades,
**I want** poder etiquetar mis actividades con habilidades y que el tiempo se acumule automáticamente,
**so that** pueda ver cuánto tiempo real he invertido en cada competencia sin ingreso manual.

## Acceptance Criteria

### AC1 — Server Action `tagActivityWithSkill`
**Given** una actividad (stepActivity) y una skill del usuario,
**When** el usuario llama a `tagActivityWithSkill(stepActivityId, skillId)`,
**Then** se inserta en `step_skill_tags` con (stepActivityId, skillId, userId),
Y retorna `{ success: true }`.

### AC2 — Guard: ownership en `tagActivityWithSkill`
**Given** cualquier llamada a `tagActivityWithSkill`,
**When** la activity o la skill no pertenecen al usuario autenticado,
**Then** retorna `{ success: false, error: 'No autorizado' }`.

### AC3 — Guard: duplicado en `tagActivityWithSkill`
**Given** una activity ya tiene la skill etiquetada,
**When** se intenta etiquetar de nuevo,
**Then** retorna `{ success: true }` (idempotente — no error).

### AC4 — Server Action `removeSkillTag`
**Given** un tag existente en `step_skill_tags`,
**When** el usuario llama a `removeSkillTag(stepActivityId, skillId)`,
**Then** se elimina el tag y retorna `{ success: true }`.
Y si el tag no existe o no pertenece al usuario, retorna `{ success: false, error: 'Tag no encontrado' }`.

### AC5 — Actualización automática de `timeInvestedSeconds` al parar timer
**Given** un timer activo sobre una activity que tiene skills etiquetadas,
**When** el timer se para (stopTimer action — Story 5.8),
**Then** se suman los `durationSeconds` del time entry completado a `skills.timeInvestedSeconds` de cada skill etiquetada en esa activity.

### AC6 — Query `getSkillsForActivity`
**Given** una activity con skills etiquetadas,
**When** se llama a `getSkillsForActivity(stepActivityId, userId)`,
**Then** retorna array de Skills con sus datos actualizados.

### AC7 — UI: selector de skills en detalle de activity (Calendario)
**Given** el usuario ve una activity en el Calendario (vista Día),
**When** la activity no tiene skills etiquetadas,
**Then** muestra un selector "+ Etiquetar habilidad" con dropdown de sus skills activas.
Y cuando hay skills etiquetadas, las muestra como badges con botón "×" para eliminar.

### AC8 — `timeInvestedSeconds` visible en Skills list (Story 7.1)
**Given** una skill con tiempo acumulado,
**When** se renderiza la Skills list,
**Then** muestra el tiempo formateado: "142h 30min" o "0h" si no hay tiempo.

### AC9 — Tests ≥ 531 + nuevos de Story 7.2
**Given** la implementación completa,
**When** se ejecuta `npm test`,
**Then** todos los tests pasan (≥ 531 anteriores + nuevos de Story 7.2).

## Risks

- **Race condition en stopTimer:** Si múltiples time entries terminan casi simultáneamente (edge case), la suma podría duplicarse. Mitigación: usar `+= durationSeconds` vía SQL directo (`skills.time_invested_seconds = skills.time_invested_seconds + X`) en lugar de read-modify-write desde Drizzle.
- **Activity sin skills etiquetadas:** stopTimer no falla si no hay skill tags — simplemente no actualiza nada. Cero regressions en timer.
- **Skills archivadas:** Si una skill fue archivada pero tenía tags, `stopTimer` igual la actualiza (se preserva el historial). Comportamiento intencional.

## Dev Notes

### Integración con `stopTimer` (actions/timer.ts)

Paso extra al final del `stopTimer`, DESPUÉS del update del time entry:

```typescript
// Después de db.update(timeEntries).set({ isActive: false, endedAt, durationSeconds })

// Propagate time to tagged skills
const tags = await db
  .select({ skillId: stepSkillTags.skillId })
  .from(stepSkillTags)
  .where(eq(stepSkillTags.stepActivityId, entry.stepActivityId))

if (tags.length > 0) {
  await db.execute(sql`
    UPDATE skills
    SET time_invested_seconds = time_invested_seconds + ${durationSeconds},
        updated_at = NOW()
    WHERE id = ANY(${tags.map(t => t.skillId)})
      AND user_id = ${userId}
  `)
}
```

Usar `sql` template de Drizzle para el raw increment.

### Función pura `formatSkillTime`

```typescript
export function formatSkillTime(seconds: number): string {
  if (seconds === 0) return '0h'
  const hours = Math.floor(seconds / 3600)
  const minutes = Math.floor((seconds % 3600) / 60)
  if (minutes === 0) return `${hours}h`
  return `${hours}h ${minutes}min`
}
```
